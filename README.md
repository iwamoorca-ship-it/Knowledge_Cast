# KnowledgeCast

## Overview

KnowledgeCast is a Proof of Concept (PoC) for an AI-powered podcast generation system.

The goal of this project is to automatically:

1. Collect business news
2. Generate knowledge-oriented summaries
3. Convert the summaries into speech
4. Publish podcast episodes through an RSS feed
